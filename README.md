# Autism Spectrum Disorder (ASD) Classification

This project builds a machine learning pipeline for predicting **ASD traits** using clinical, behavioral, and demographic data. It includes dataset extraction from a ZIP file, data cleaning, missing value imputation, categorical encoding, numerical scaling, and train-test splitting to prepare the data for classification models.

## Features
- ZIP file extraction and CSV loading
- Dropping irrelevant ID columns
- Handling missing values
- Encoding binary and categorical features
- Scaling numerical features
- Preparing train/test data for ML models

## Tech Stack
Python, Pandas, NumPy, Scikit-learn

## Target
`ASD_traits`

## Workflow
1. Extract dataset from ZIP  
2. Load and clean data  
3. Preprocess features  
4. Split into train and test sets  
5. Prepare data for classification modeling
